# X-IDE Project Roadmap

## Phase 1: Foundation & Core Graphics Engine (Completed)
- [x] **Project Setup**: Initialize Next.js, Tailwind CSS, and basic project structure.
- [x] **Canvas Implementation**: Set up React-Konva for the main design area.
- [x] **Basic Shapes**: Implement Rectangle and Path (Pen tool) drawing.
- [x] **Selection & Manipulation**: Implement selection, dragging, and resizing of shapes.
- [x] **Undo/Redo System**: Implement a history stack for undo/redo functionality.
- [x] **Grouping & Frames**: Implement grouping of shapes and "Frame" behavior (clipping).
- [x] **Snapping & Alignment**: Implement smart guides for snapping to other objects.
- [x] **Artboards**: Implement Artboard support with isolated coordinate systems and clipping.
- [x] **Typography System**: Implement Text tool, font properties, and inline editing.

## Phase 2: Advanced Features & UI (Completed)
- [x] **Layer Management**: Enhance the layer tree with visibility toggles, locking, and renaming.
- [x] **Properties Panel**: Expand the properties panel to support all shape types and properties (fills, strokes, effects).
- [x] **Asset Management**: Implement image uploading and management (Assets Panel).
- [x] **Components & Symbols**: Implement reusable components (Components Panel).

## Phase 3: Code Generation & Export (Completed)
- [x] **Tailwind Code Gen**: Refine the Tailwind CSS code generation to be more robust and accurate.
- [x] **React Component Export**: Allow exporting designs as React components.
- [x] **Image Export**: Export designs as PNG/JPG (1x/2x).
- [x] **Preview Mode**: Implement a preview mode to see the design as a live web page.

## Phase 4: Polish & Optimization (Completed)
- [x] **Performance Optimization**: Optimized rendering with React-Konva and memoization.
- [x] **Keyboard Shortcuts**: Implement comprehensive keyboard shortcuts (Undo/Redo, Group/Ungroup, Copy/Paste, Delete).
- [x] **Zoom & Navigation**: Implemented Zoom UI and Pan controls.
- [x] **Prototyping**: Implemented basic interaction (click-to-navigate) for Artboards.
- [x] **Vector Engine**: Upgraded to SVG Path support for advanced vector editing.
- [ ] **Dark/Light Mode**: (Future) Ensure full support for dark and light modes.
